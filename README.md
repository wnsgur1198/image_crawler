## Usage example

원하는 이미지에 대해 키워드를 사용하여 구글 이미지를 검색하고 캡처하는 방식으로 이미지를 수집하는 프로그램이다.

단, 유효한 이미지 캡처 개수는 약 50개이다. 그 이상으로 지정하면 하얀색 이미지가 덮어져서 다운로드되기 때문에 원하는 이미지를 얻을 수 없다.
